# Demographic Feature Analysis

Author: Bereket Tarekegn

This folder contains my demographic feature analysis for the ADS 503 diabetes prediction group project.

## Feature Group

The demographic feature group includes Age, Sex, Education, Income, AnyHealthcare, and NoDocbcCost.

## Goal

The goal is to evaluate how demographic and socioeconomic variables help predict diabetes status.

## Models

The planned models are multinomial logistic regression, decision tree, and random forest.
